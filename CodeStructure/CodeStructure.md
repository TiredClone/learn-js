# Инструкции

Инструкции - это синтаксические конструкции и команды, которые выполняют действия.
Мы уже видели инструкицю `alert('Привет, мир!')`, которая отображает сообщение "Привет, мир!".
В нашем коде может быть столько инструкций, сколько мы захотим. Инструкции могут отделяться точкой с запятой.

Здесь мы разделили сообщение "Привет, мир!" на два вызова `alert`:

```js
alert('Привет'); alert('Мир');
```

Обычно каждую инструкцию пишут на новой строке, чтобы код было легче читать:

```js
alert("Привет");
alert("Мир");
```

## Точка с запятой

В большинстве случаев точку с запятой можно не ставить, если есть переход на новую строку.

```js
alert("Привет");
alert("Мир");
```

В этом случае JavaScript интерпретирует перенос строки как "Неявную точку с запятой. Это называется **автоматическая вставка точки с запятой**.

В некоторых ситуациях новая строка всё же не означает точку с запятой. Например:

```js
    alert(3 +
        1
        + 2);
```

Код выведет 6, потому что JavaScript не вставляет здесь точку с запятой. Интуитивно очевидно, что если строка заканчивается знаком `+`, значит, это "незавершённое выражение", поэтому точка с запятой не требуется.

### Пример ошибки

```js
alert("Hello");

[1, 2].forEach(alert);
```

Выводится Hello, затем 1, затем 2. Если убрать, получится:

```js
alert("Hello")[(1, 2)].forEach(alert);
```

Пропала только точка с запятой. Если запустить, выведется только первый `alert`, а затем мы получим ошибку. Это потому что JavaScript не вставляет точку с запятой перед квадратными скобками [...]. Код выполняется как одна инструкция.

```js
alert("Hello")[(1, 2)].forEach(alert);
```

## Комментарии

Комментарии могут находится в любом месте скрипта. Они не влияют на его выполнение, посколько движок просто игнорирует их.

Однострочные комментарии начинаются с двойной косой черты `//`. Комментарий может как занимать строку целиком, так и находится после инструкции.

Многострочные комментарии начинаются косой чертой со звёздочкой `/*` и заканчиваются звёздочкой с косой чертой `*/`.

**Вложенные комментарии не поддерживаются!**
